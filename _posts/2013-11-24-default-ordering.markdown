DEFAULT ORDERING

    
     ASC / By Default
     - Numeric values --- ( 1 to 999 ).
     - Character 	  --- Alphabetical order.
     - Date values    --- earliest value first, such as 01-JUN-93 before 01-JUN-95.
     - Null (empty) values are displayed
